Web Diagnostics for Imperal Cloud — DNS, SSL, HTTP, email, blacklists, ports, SMTP, geo probes,
SEO, network diagnostics, automated monitoring. Competes with DNSChecker.org + MXToolbox + IntoDNS,
with unique multi-region probes (EU/US/SG/MD simultaneously) and AI-powered analysis.

## Complete function reference — 31 functions

### On-demand diagnostics
dns_lookup — A/AAAA/IP/MX/NS/TXT/CNAME/SRV/DNSSEC/all; propagation across 6 global servers
  (Google 8.8.8.8, Cloudflare 1.1.1.1, Quad9, OpenDNS, Yandex, 114DNS China); authoritative NS
ssl_check — grade A-F, expiry days, issuer; full=chain+SANs+TLS versions+fingerprint
whois_lookup — domain (registrar/dates/nameservers/status) or IP (ASN/org/country/network range)
http_check — security headers grade A+ to F, missing headers + recommendations, redirect chain,
  HTTP status code, response time
email_check — SPF/DMARC/DKIM/BIMI individual + full grade A-F; generate SPF/DMARC records;
  trace raw email headers to find originating IP and server path
blacklist_check — IP: 30 DNSBL (Spamhaus ZEN/SBL/XBL, SpamCop, Barracuda); domain: IP DNSBL + SURBL
  verdict: clean / listed / critical
port_scan — single TCP port check or presets: web(80/443/8080/8443), mail(25/587/465/993/995),
  database(3306/5432/6379), all — parallel scan
smtp_test — real SMTP handshake on 587/25/465 via MX or direct host; STARTTLS, AUTH methods,
  server banner, software detection
geo_check — EU/US/SG/MD simultaneous probes: dns (mismatch/anycast detection), ping (latency),
  http (availability+latency), ssl (cert validity per region), traceroute (full MTR per region),
  full (dns+http+ssl all at once — 12 parallel calls)
network_check — ping ICMP, traceroute EU/NL origin, reverse DNS (PTR), ip_lookup (geo+ASN+org),
  ip_lookup_quick (country+org only), asn (ASN WHOIS + announced prefixes)
seo_check — meta tags (title/description length validation + SEO issues), robots.txt parsing,
  sitemap.xml validation + URL count, Google indexing status
domain_full_check — all checks in parallel (semaphore 5): dns+ssl+whois+http+email+blacklist+geo

## Interpreting results — always be actionable

DNS: No MX records = email is broken. Propagation inconsistent = change still propagating (check TTL).
SSL: Expired or invalid = site showing security errors. Days < 14 = urgent renewal. Grade C/D = weak config.
HTTP: Grade F = HSTS missing (clickjacking) + CSP missing (XSS). Show which headers are missing.
Email: SPF missing = anyone can spoof domain. DMARC p=none = not enforcing, only monitoring.
  DKIM missing = delivery issues + spam risk. Grade C/D = at least one critical record missing.
Blacklist: critical = Spamhaus ZEN/SBL/SpamCop/Barracuda = email severely blocked, fix immediately.
  listed (non-critical) = some recipients affected, monitor and investigate.
SMTP: Port 587 down = outbound mail broken. No STARTTLS = unencrypted mail transmission.
Geo: EU ok, US down = regional routing issue. DNS mismatch = CDN or anycast misconfiguration.
Propagation: Inconsistent = DNS change propagating — depends on TTL, wait or lower TTL first.
Ports: 3306/5432/6379 open externally = critical security risk — firewall immediately.

## Common diagnostic workflows

Email not delivering → email_check(full) + blacklist_check(domain) + smtp_test
DNS change check → dns_lookup(propagation) — Consistent across all 6 servers means done
New domain audit → domain_full_check(dns+ssl+http+email+blacklist)
Site down in region → geo_check(http) — shows which of EU/US/SG/MD can reach site
IP reputation → blacklist_check(ip) + network_check(ip_lookup) + network_check(reverse_dns)
Set up monitoring → create_monitor_full(name, domains, ["ssl","http","email","blacklist"], 24)
Security audit → http_check(missing) + ssl_check(full) + port_scan(all)
